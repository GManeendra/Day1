What is Meant by the HTML ?

* HTML provides the structure and content of the web page by using Tags.
* It is used to create the webpages.
* it is used for backbone of the websites.

History of HTML ?

* It is developed by "Tim berners-lee" in 1989 and it is officially released on 1991.

Version of HTML ?

* 1991 - HTML 1.o - It only the content is displayed.
* 1995 - HTML 2.o - In this version Add the more tags.
* 1997 - HTML 3.2 - In this version Add the more tags.
* 1999 - HTML 4.o - In this version add like (registration forms..... etc).
* 2014 - HTML 5 - it is the latest version.

Naming conventions ?

Rules

* do not start with the fileName with a number.
* do not use spaces in fileName.
* do not use special character in fileName.
* do not use filename as predefined keyword.
* use all upper case Letters.
* use underscore while using fileName.
* use hyphen instead of spaces.

core components or build bolck elements ?

* Tags
* attribtes
* Elements

TAGS

* Tags is nothing but the predefined keywords.
* Each keyword has induilt properties to displayed on the webpages.
* Tags are used for structure of the webpages.

Syntax : < TagName > < /TagName >

Example:

*< HTML > < /HTML >.....etc

TYPES of Tags :

* Single tags :

Example:

*< img src="" alt="" class="">
*< br >
* < hr >....etc

* Paired Tags :

Example:

*< p>< /p>
* < b>< /b>....etc

ATTRIBUTES

* Attribtes are used to add extra information to the tags.
* Attribtes are modify the inbuilt properties of the tag.

Syntax :

< TagName AttributeName="AttributeValue">< /TagName>
< TagName AttributeName="AttributeKey : AttributeValue; "> < /TagName>

TYPES of Attributes :

* Global Attribtes
* Element Specific Attribute

Global Attribtes

Example:

* Style
* Class
* id ....etc

Element Specific Attribtes

Example:

* Src
* Alt
* href ....etc

ELEMENTS

* Element are the building block of the web pages.

Syntax : < TagName AttributeName="AttributeValue"> Content < /TagName>

* Block level Elements
* Inline Elements
